Game called "Ahoy" — a pirate/nautical game inspired by Sid Meier's Pirates and Dwarf Fortress.

**Vision:** Pirates-style gameplay (exploration, combat, trade, reputation) combined with a living, dynamic world simulation (Dwarf Fortress-style emergent behavior, world state, factions with agency).

**Approach:**
- Backend-first: focus on core systems and behaviors before committing to a frontend
- Frontend-agnostic design: architecture should allow a rendering layer to be slotted in later
- Language: C#
- Design document is being built iteratively before full system specs are written

**Why:** User wants to nail the simulation/behavior layer before locking in any UI/rendering technology.
